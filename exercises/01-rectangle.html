01 - Rectangle Exclusions

Exclusions allow you to define areas for text to wrap around. As one example, you can define a rectangle which text will avoid. In order to layout text this way, you need to calculate the intersection between each line and the rectangle. A line is represented by the minimum and maximum y value for the line. A rectangle is represented by an x, y, width, and height. An intersection is represented by the minimum and maximum x value that is inside both the line and rectangle bounds.
